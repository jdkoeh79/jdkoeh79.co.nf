THE THREE LAYERS OF WEB DESIGN

Web design is based on three main 'layers': Content, presentation and behavior.

Content: HTML is where the text and images on your page are entered.

Presentation: CSS controls the color, size, spacing, positioning and all other aspects of styling applied to your content.

Behavior: JavaScript listens for actions such as mouse clicks or even mouse pointer hover and changes aspects of your page (can alter your HTML and/or your CSS code) to make it come alive.

FREE ONLINE CODE SCHOOLS

Like he says at the beginning of the video (even though it's now 2016), no matter what path you specialize in, you're going to want to start with HTML, CSS, and Javascript.

You can jump right in at any of the free learn-to-code sites without having to download or install anything. I suggest doing a few lessons at a few different sites to get a feel for their format and although you could pick one you like and go with it, going through the lessons at more than one site will give you extra practice to help embed the concepts into your long-term memory.

I recommend starting off at Freecodecamp and Codeacademy. If/when you get stuck, I found it immensely helpful to hear/see things be explained in Udacity's videos. After you've taken Freecodecamp and Codeacademy for a test-drive, do check out Khan Academy.

Codecademy

All of the lessons are free, you only pay for their quiz's. I do lessons here for extra practice. Lessons are text instructions with an in-site editor that you code in.

Even if you aren't gonig to check this one out right away, go register and in the account options at the bottom, make sure you get the emails from Quincy Larson. He shares must-read articles and they are usually really interesting at worst.

Freecodecamp

This site is completely free and is set up as a more linear path with over 2000 hours of lessons for both Front End and Back End development. Toward the end, you help on real projects for non-profit organizations to give you some real world experience. They also have their own chat and forums for reaching out to fellow coders when you get stuck and there are even local Facebook groups, although our Oklahoma City and Tulsa groups aren't very active.

Udacity

All courses are free but you can pay for a 'Nanodegree' program. Structured more like legit online college courses, some courses list pre-requisites and each course is comprised of mini lectures by an instructor via videos where they explain things using a digital whiteboard. They range from 1 to 7 minutes in length and I, personally, have been very successful with this method of learning. Courses are broken down into lessons and each lesson has steps you go through that may just be a video, may be a quiz and sometimes end with you writing code in their in-site editor. Lessons are then followed up by practice sets and more challenging optional practice sets. The instructors in the few courses I've taken all did an excellent job at presenting their topics.

Khan Academy

This should be the one on the list you've already heard of. Famous for their math videos, they now cover a wide variety of topics including web development and computer programming. Completely free as far as I know. Their format is very unique, using 'codeing talk-throughs' you can actually pause their videos (which are more like recordings with voice-over of someone working in an on-site text editor) and pick up where they left off, typing into the same editor box where you see them type when you hit the play button. Very cool.

Flatiron School

Found this in a Quora article. They offer one free course, two to choose from; JavaScript or Ruby. The JavaScript course is 50+ hours of coding lessons. Might be worth checking out.

Codeschool

A Pluralsight company. The first step in each course is free, subscription is required to unlock the rest. Monthly plan: $29/mo, Yearly plan $19/mo. It's been a while since I've done any lessons here but if I recall correctly, they use a combination of videos and text instructions with an in-site editor.

Treehouse

7 day free trial and then $25/mo. I did a trial but I don't remember what their course format was like. They also offer a Techdegree plan at $199/mo, "For anyone who wants to earn a certification and launch a career". The above link will take you to their 'Pick Your Plan' page with a breakdown of what each one includes.

Pluralsight

Looks to be geared more toward businesses looking to improve the skillset of their employees but there is also a membership for individuals thats $29/mo or $299/year with a free trial.

MIT Open Coarse Ware

These are full fledged free versions of online MIT courses. 40 minute lecture videos, etc. Hardcore, but it's MIT, so I had to include it in the list. Link to their Intro to Computer Science and Programming course (which like Udacity, uses Python to introduce basic computer programming concepts that you will use while writing JavaScript).

TEXT EDITORS

Typically, code is typed up in a text editor and saved with a file extension representing the language that the file contains:

File Type Extension
Webpage .html
CSS Style Sheet .css
JavaScript File .js

Think Notepad on steroids. The latest and greatest boast features such as auto-complete, multi-pane views, project/folder sidebars, and extenstions that add countless other features to help boost your productivity and accuracy. My absolute favorite text editor is Sublime Text 3 (the snippet of code in this page's header is a Sublime Text 3 screenshot).

Stay tuned for a list of my favorite Sublime Text 3 addons.

A few other popular text editors are:

Brackets
  • "A modern, open source text editor that understands web design."
  • Made with JavaScript
  • Extension Support
  • Themes
  • Open Source
Atom
  • "A hackable text editor for the 21st Century"
  • a desktop application built with HTML, JavaScript, CSS, and Node.js integration
  • Built-in Package Manager
  • Themes
  • Features include: Smart Autocomplete, File system browser, multiple panes, Find and replace
  • Open Source
Notepad++
  • "Notepad++ is a free (as in "free speech" and also as in "free beer") source code editor..."
  • written in C++ and uses pure Win32 API and STL which ensures a higher execution speed and smaller program size
  • Completely Free
